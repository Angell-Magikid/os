## 权限层级
- CEO: 全局管理、权重设置、审批所有
- 总监级(COO/CTO/CFO/总监): 项目发起、团队管理
- 经理级(IT/媒体/课程): 项目参与、管理
- 成员级(开发): 项目申报、进展记录

## 功能模块
1. 人员管理 - 档案、登录、HR信息
2. 项目管理 - 申报、加入、进度（≤1个月）
3. 优先级矩阵 - 重要×紧急
4. AI参与追踪 - 人/AI比例
5. 自动报告 - 项目报告+周报
6. 关系图谱 - 项目人员网络

## 架构
- 前端: HTML5 + CSS3 + Vanilla JS
- 数据: LocalStorage (本地存储)
- 可扩展: 预留API接口
